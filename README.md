# ipGenz 🚀

ipGenz is a modern, blazing-fast, and premium IPTV platform built with Next.js and NestJS. It provides an unparalleled streaming experience for Live TV, Movies, and TV Series, featuring an elegant UI/UX and robust backend synchronization.

## ✨ Features
- **Premium User Interface**: Dark mode, glassmorphism, smooth animations, and a responsive design.
- **Global Sync Engine**: Seamlessly fetches and categorizes M3U and Xtream Codes playlists.
- **Smart Caching**: Auto-removes stale duplicates and caches metadata instantly.
- **Advanced Player**: Integrated high-performance video player supporting various formats.
- **Admin Dashboard**: Real-time metrics, user management, and provider configuration.
- **Demo Mode**: Built-in "Experience the App" public demo feature.

## 🛠 Tech Stack
- **Frontend**: Next.js 15, React, TailwindCSS, TypeScript
- **Backend**: NestJS, Prisma ORM, PostgreSQL
- **Media**: FFmpeg, TMDB API (for rich metadata)

## 🚀 Deployment (100% Free)
1. **Frontend**: Deploy to Vercel. Set `NEXT_PUBLIC_API_URL` to your backend URL.
2. **Backend**: Deploy to Render (Web Service).
   - Build Command: `npm install && npx prisma generate && npm run build`
   - Start Command: `npm run start:prod`
3. **Database**: Use Neon.tech or Supabase for a free PostgreSQL database. Set `DATABASE_URL` in Render.
